# 📰 El Diario de Santiago — Newsletter Diaria Personal

Periódico digital de consulta diaria, personalizado para un cardiólogo electrofisiólogo en formación, inversor particular y deportista basado en evidencia científica.

## 🗂️ Estructura del repositorio

```
📁 Newsletter-diaria.-
├── README.md                  → Este archivo
├── prompt/
│   └── prompt_agente.md       → Prompt completo del agente
├── plantillas/
│   └── plantilla_diaria.md    → Plantilla de salida de cada edición
└── ediciones/
    └── YYYY-MM-DD.md          → Una edición por día (generadas por el agente)
```

## ⚙️ Cómo usar

1. Copia el contenido de `prompt/prompt_agente.md`
2. Pégalo en tu agente favorito con acceso a herramientas web (Perplexity, ChatGPT, n8n, etc.)
3. El agente generará una edición diaria siguiendo la plantilla de `plantillas/plantilla_diaria.md`
4. Guarda cada edición en `ediciones/YYYY-MM-DD.md`
